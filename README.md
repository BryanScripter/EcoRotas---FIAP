# 🌱 EcoRotas - Mobilidade Sustentável

Aplicativo Android desenvolvido para incentivar escolhas de mobilidade mais sustentáveis, permitindo que o usuário compare a emissão de CO₂ entre diferentes meios de transporte.

---

## 🎯 Objetivo

O **EcoRotas** tem como principal objetivo conscientizar usuários sobre o impacto ambiental de suas escolhas no dia a dia.

A aplicação calcula e compara a emissão de dióxido de carbono (CO₂) com base na distância de um trajeto, ajudando o usuário a identificar a alternativa mais sustentável.

---

## 🌍 Contexto ESG

O projeto está diretamente ligado ao pilar **Environmental (E)** do conceito ESG.

Através da comparação de emissões de carbono, o aplicativo incentiva:

* Redução da poluição
* Uso de transportes sustentáveis
* Consciência ambiental no cotidiano

## 🚀 Funcionalidades

### 🔐 Autenticação

* Cadastro de usuários
* Login com validação
* Armazenamento local de dados

### 🏠 Home

* Saudação personalizada
* Acesso rápido às funcionalidades
* Exibição de CO₂ economizado

### 📍 Cálculo de Rotas

* Definição de origem e destino
* Cálculo de impacto ambiental
* Comparação entre transportes

### 🚲 Comparação de Transporte

* Carro
* Ônibus
* Bicicleta

Cada opção apresenta:

* Emissão estimada de CO₂
* Tempo de deslocamento

### 📊 Histórico

* Registro das rotas realizadas
* Distância percorrida
* CO₂ economizado

### 🌱 Resultado Sustentável

* Indicação da melhor opção de transporte
* Economia de CO₂ em relação ao carro
* Feedback incentivando escolhas sustentáveis

## ⚠️ Observações

* A integração com API de mapas (Google Maps) não foi ativada devido à necessidade de chave com cobrança
* Algumas telas foram implementadas como protótipo por conta dessa limitação
* Os cálculos de CO₂ são baseados em estimativas médias por km

---

## 🔌 Integração com API

O projeto foi planejado para utilizar a API do Google Maps para cálculo de rotas.

👉 Para ativar:

* Adicionar uma API Key no arquivo `local.properties`
* Configurar a integração com o serviço
